### Usage

```
./start.sh WEBPACK_VERSION MODE BOOTSTRAP_VERSION
```

where `WEBPACK_VERSION` - `3` or `4`, `MODE` - `production` or `development`, `BOOTSTRAP_VERSION` - `3` or `4`.
